Habilidades Técnicas

Un resumen de mis competencias desarrolladas a lo largo de mi formación académica.

Bases de datos

Experiencia en el diseño y gestión de bases de datos relacionales, principalmente con MySQL, aplicando consultas y estructuras eficientes para el manejo de información.

Desarrollo web

Desarrollo de interfaces web utilizando HTML, CSS y JavaScript, asegurando una estructura clara, estilos consistentes y funcionalidad básica.

Programación POO

Aplicación de principios de Programación Orientada a Objetos para la creación de sistemas estructurados y escalables.

Redes y sistemas

Conocimientos fundamentales en configuración de redes, diagnóstico básico y mantenimiento de sistemas informáticos.

Diseño y maquetación

Desarrollo de interfaces visuales y maquetaciones estéticas, aplicando paletas de colores pastel y diseños coherentes con estilo moderno.

Control de versiones

Uso de Git y GitHub para la gestión de versiones, manejo de repositorios y control del flujo de trabajo en proyectos individuales.

Circuitos eléctricos y electrónicos

Diseño y simulación de circuitos eléctricos y electrónicos, integrando sensores, componentes y prácticas enfocadas en funcionamiento real. Implementación de sistemas automatizados a pequeña escala, integrando sensores para control de iluminación, humedad y simulaciones como bombeo de agua.

Arquitectura de computadoras

Conocimientos sobre la estructura y funcionamiento interno del hardware, componentes esenciales y ciclos de procesamiento computacional.
